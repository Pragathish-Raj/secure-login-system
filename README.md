Secure Login System is a full-stack authentication application built using Node.js, Express, and MongoDB.
The system provides secure user registration, login authentication, password encryption, and email-based password recovery with modern security practices.

Features:

-> User Registration and Login

-> Password Encryption using bcrypt

-> JWT Authentication for secure sessions

-> Forgot Password with Email Verification

-> Secure Backend API with Express

-> MongoDB Database Integration

-> File upload support using Multer

requirements :

-> bcrypt==6.0.0

-> cors==2.8.6

-> dotenv==17.3.1

-> express==5.2.1

-> jsonwebtoken==9.0.3

-> mongoose==9.3.0

-> multer==2.1.1

-> nodemailer==8.0.2
